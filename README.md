# tfm-youtube-sentiment-pipeline
Pipeline ETL escalable para la extracción oficial de datos de YouTube mediante YouTube Data API v3, procesado en Azure Functions y almacenado en Azure Blob Storage para análisis de sentimiento.

## Instrucciones
1. Activar el entorno virtual:
bash
2. source .venv/Scripts/activate
Arrancar el servidor de Azure:
bash
func start
